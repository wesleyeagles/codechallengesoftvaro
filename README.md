## Requirements

* Node 14.17.0
* Newsapi Key
* Google API ID

## Installation

1. Clone the project

```
$ git clone https://github.com/wesleyeagles/codechallengesoftvaro.git
```
2. Set environment variables on .env.local file, variables names in the file env.local.example
3. Install libraries

```
$ yarn # or npm install
```

4. Run the project
```
$ yarn run dev # or npm run dev
``` 

## Tests

```
$ yarn test
```
